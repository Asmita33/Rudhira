# Rudhira - A Social Service App
Rudhira is a Social Service App to connect blood donors and seekers through an efficient and verified process.

#### App's APK : https://drive.google.com/file/d/1IVwVAC984Imyp4I467AGJo1YzmzU2r_y/view?usp=sharing
#### Github repo URL : https://github.com/Asmita33/Rudhira
#### Demo Video: https://drive.google.com/file/d/1rOp-x2-2UlJp0dUPhPec4cu0VcAEjQXo/view?usp=sharing
## App Basic Features:
1. Option to raise a blood request:
2. Blood group
3. Amount of blood required
4. Hospital location
5. Patient condition
6. Display list of blood requests
7. Admin login for handling emergency cases and verification of documents.
8. Message/Email notification to admin after a request has been registered.
9. The request will be displayed in the blood request list only after the admin’s approval.
10. Valid documents to be submitted before requesting blood donation (so that there isn’t an illegal purchase of blood).
11. Blood donors and blood seekers should be connected independently via a chat-based system.
12. Emergency button feature for priority treatment.
13. Cooldown feature after a successful donation.
14. The seeker should be able to cancel the request once no longer required.
15. Notification to registered users (if enabled) of Blood donation camps.

## App Advance Features:
1. Notification to all registered users after admin’s approval if the request raised is within the area limit.
2. Notification to request raiser via Email, messaging if the request has been accepted.
3. Sorting the blood requests on the basis of distance/time/patient’s condition.
4. Map feature to get hospital locations(provided by seeker) & directions and to see nearby blood banks/hospitals/camps.
5. Option to share any particular blood request with others on different apps.
6. Multiple language support.
7. Search feature for particular address requests / particular person’s requests.
8. History of user’s donated/raised requests.
